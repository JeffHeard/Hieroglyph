Graphics.Rendering.Hieroglyph.Cairo
Description

Cairo backend to Hieroglyph. Future plans include making StateModifier part of the base distribution and making Frame the basis for all backends to Hieroglyph. The real challenge will be making all implementations look the same.

Note in particular that pattern functionality and Pango layouts and font rendering engines are not implemented yet. This is both because these features are fairly complicated, and because I haven't figured out yet how I might make them portable to OpenGL using textures and FTGL. if someone wants to take this task on, I'd be pleased as punch.
