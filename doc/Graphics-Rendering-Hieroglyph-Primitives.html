Graphics.Rendering.Hieroglyph.Primitives
Description

This is Hieroglyph, a 2D scenegraph library similar in functionality to a barebones stripped down version of Processing, but written in a purely functional manner.

See individual implementations (like the Graphics.Rendering.Hieroglyph.Cairo module) for more information on how to use this library.
